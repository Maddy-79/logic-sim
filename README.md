# Logic Gates Simulator

A browser-based logic circuit simulator.

## Features

- AND
- OR
- NOT
- NAND
- NOR
- XOR
- Multi-input gates
- Live simulation
- Undo/Redo
- Keyboard shortcuts
- Responsive UI

## Tech Stack

- React
- React Flow
- TypeScript
- Vite
- Cloudflare Pages
